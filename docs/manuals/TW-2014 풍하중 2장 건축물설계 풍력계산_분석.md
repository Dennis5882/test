# TW-2014 풍하중 2장 건축물설계 풍력계산 분석

## 1. 結構力學核心邏輯
本章規定建築物設計風力之計算方法，涵蓋封閉式、部分封閉式與開放式建築物主要風力抵抗系統的設計風壓與風力公式（2.1~2.4節），以及簡化計算方法（2.13節）。風速垂直分布採用指數律公式（2.5節），地況分為A、B、C三類。基本設計風速依臺灣地區分區給定（2.4節）。

## 2. 主要公式與標準
- **設計風壓/風力公式**：
  - 封閉式/部分封閉式：p = qGCp - qi(GCpi) (2.1)
  - 開放式：pp = qp(GCpn) (2.3)
  - 主要風力抵抗系統：F = q(z_Ac) G Cf Ac (2.4)
- **風速垂直分布**：Vz = V10 (z/10)^α (2.5)
- **風速壓**：q(z) = 0.06 K(z) K_zt [I V_10(C)]^2 (2.6)
- **陣風反應因子**：
  - 一般建築物：G = 1.88 或式(2.9)
  - 柔性建築物：G = 1.927 (1 + 1.7 I_z √(g_Q^2 Q^2 + g_R^2 R^2)) / (1 + 1.7 g_V I_z) (2.13)
- **橫風力**：
  - 圓形斷面：W_{rz} = 0.28 ρ U_{rD}^2 C_r A (C2.16)
  - 矩形斷面：WLz = 0.87 (L/B) WDz (2.21) 或式(2.22)
- **扭轉力矩**：M_{Tz} = 0.28 B W_D^*(z) (2.23) 或式(2.24)
- **低層建築物簡化公式**：
  - 順風力：S_{Dz} = 1.49 I V_{10}(C) K_{zt}(h) λ^2 A_z (2.25)
  - 橫風力：S_{Lz} = (0.6 + 0.05 L/B) S_{Dz} (2.29)
  - 扭轉力：S_{Tz} = 0.21 B S_{Dz}^*(z) (2.30)

## 3. 數據流向圖
```
輸入：基本設計風速 V10(C)、建築物幾何尺寸、地況類別、用途係數 I
│
├─ 計算風速垂直分布 Vz (2.5)
│  └─ 計算風速壓 q(z) (2.6) 含地況係數 K(z) 及地形係數 Kzt (2.8)
│
├─ 判斷建築物類型：
│  ├─ 一般建築物 (fn > 1 Hz)：G = 1.88 或查表 C2.9
│  └─ 柔性建築物 (fn ≤ 1 Hz)：計算 G (2.13) 含背景響應 Q 及共振響應 R
│
├─ 選擇風壓係數 Cp 或風力係數 Cf (2.8節)
│  └─ 內風壓係數 GCpi (2.9節)
│
├─ 計算設計風壓/風力：
│  ├─ 封閉式：p = qGCp - qi(GCpi) (2.1)
│  ├─ 開放式：pp = qp(GCpn) (2.3)
│  └─ 主要系統：F = q G Cf Ac (2.4)
│
├─ 若為低層建築物 (h<18m, h/BL<3, 0.2≤B/L≤5)：
│  ├─ 順風力 S_Dz (2.25)
│  ├─ 橫風力 S_Lz (2.29)
│  └─ 扭轉力 S_Tz (2.30)
│
├─ 若為高層建築物：
│  ├─ 橫風力 WLz (2.21) 或 (2.22)
│  └─ 扭轉力矩 M_Tz (2.23) 或 (2.24)
│
└─ 風力組合 (2.12節)：SRSS 或 100%/30% 規則
```

## 4. 批判性審查與疑慮
- **邊界條件**：
  - 過渡地況的風速垂直分布如何合理分析？
  - 鄰近建築物遮蔽效應的折減如何量化？
  - 地形係數 Kzt 計算條件不滿足時的合理假設？
  - 一般建築物與柔性建築物的邊界 (fn=1Hz) 是否明確？
  - 陣風反應因子近似式 (2.13) 適用條件 (1≤h/B≤6, 1≤B/L≤5) 外的處理？
  - 低層建築物橫風力公式 (2.29) 係數來自回歸，其他形狀適用性？
  - 扭轉力矩計算中 U* 介於 4.5~6 時的內插複雜性？
- **軟體實現衝突**：
  - 風力組合時 SRSS 方法與其他組合方法的選擇？
  - 橫風力計算中條件 (h/B<3) 不滿足時的替代方法？
  - 渦流共振及空氣動力不穩定性的額外檢查條件？
