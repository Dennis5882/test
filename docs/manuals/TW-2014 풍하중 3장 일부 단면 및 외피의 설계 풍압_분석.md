# TW-2014 풍하중 3장 일부 단면 및 외피의 설계 풍압 분석

## 1. 結構力學核心邏輯
本章規定局部構材及外部被覆物之設計風壓計算方法，適用於規則性封閉式、部分封閉式與開放式建築物或地上獨立結構物。主要計算邏輯如下：
- 封閉式或部分封閉式建築物：依高度分為≤18m及>18m兩種情況，分別使用公式(3.1)和(3.2)。
- 屋頂女兒牆：使用公式(3.3)。
- 開放式建築物斜屋頂：使用公式(3.4)。
- 外風壓係數(GCp)依高度查圖3.1或圖3.2，內風壓係數(GCpi)依2.9節。

## 2. 主要公式與標準
- 公式(3.1)：p = q(h)[(GCp) - (GCpi)]，適用於高度≤18m之封閉式或部分封閉式建築物。
- 公式(3.2)：p = q(GCp) - qi(GCpi)，適用於高度>18m之封閉式或部分封閉式建築物。
- 公式(3.3)：p = qp[(GCp) - (GCpi)]，適用於屋頂女兒牆。
- 公式(3.4)：p = q(h) G Cpn，適用於開放式建築物斜屋頂。
- 變數定義：
  - p：設計風壓
  - q(h)：屋頂平均高度處之風速壓
  - q(z)：高度z處之風速壓
  - qi：內風壓對應之風速壓
  - qp：女兒牆頂部之風速壓
  - GCp：外風壓係數（含陣風效應）
  - GCpi：內風壓係數（含陣風效應）
  - G：陣風效應因子
  - Cpn：淨風壓係數

## 3. 數據流向圖
```
Input: 建築物幾何尺寸、高度、風速、地形類別
|
+-- 判斷建築物類型：封閉式/部分封閉式/開放式
|   |
|   +-- 封閉式或部分封閉式：
|   |   +-- 高度 ≤ 18m：使用公式(3.1)
|   |   |   +-- 查圖3.1或圖3.2得GCp
|   |   |   +-- 查2.9節得GCpi
|   |   |   +-- 計算q(h)
|   |   |   +-- 輸出p
|   |   +-- 高度 > 18m：使用公式(3.2)
|   |       +-- 迎風面：q(z)隨高度變化
|   |       +-- 其他面：q(h)
|   |       +-- 查圖得GCp
|   |       +-- 查2.9節得GCpi及qi
|   |       +-- 輸出p
|   +-- 開放式建築物斜屋頂：使用公式(3.4)
|       +-- 查圖3.3得Cpn
|       +-- 計算q(h)
|       +-- 輸出p
|
+-- 女兒牆：使用公式(3.3)
    +-- 查圖得GCp
    +-- 查2.9節得GCpi
    +-- 計算qp
    +-- 輸出p
```

## 4. 批判性審查與疑慮
- 圖3.1、圖3.2、圖3.3、圖3.4、圖3.5的具體內容未在文本中提供，需查閱原圖。
- 公式(3.2)中qi的取值細節（h0z的定義）需參考2.9節。
- 開放式建築物斜屋頂的淨風壓係數Cpn具體數值需查圖3.3。
- 本規範風速平均時間10分鐘，ASCE 7-02為3秒，故(GCp)為ASCE 7-02的2.083倍，需注意轉換。
- 受風面積小於65平方公尺時適用本章，大於65平方公尺可依2.2節計算，邊界條件需明確。
